# grill-bytes
A monitoring and control device for your grill or smoker station.

## Requirements
  * 1x Particle Photon
  * 1x Nextion 3.5" Enhanced HMI Display
  * 1x Temperature Sensor (DHT11 or DHT22)
  * 2x Temperature Probes (Grill thermistor)
  * 1x 12V Blower Fan
